# 🛡️ AI Spam Email Detection
Machine Learning ka use karke spam emails ko detect karne wala ek simple tool.

## ✨ Features
* **Spam Detection:** Email content ke basis par categorize karta hai.
* **Accuracy:** Isme (aapka model name, e.g., Naive Bayes) algorithm use hua hai.
* **Easy UI:** Streamlit/Flask based interface.

## 🛠️ Tech Stack
* **Python**
* **Scikit-learn** (ML Library)
* **Pandas** (Data Handling)
